# Agent Collaboration

Before starting work in this repository:

1. Read `coop_repo/LATEST.md`.
2. Open the report linked from it.
3. Check the current worktree before editing. Do not revert another agent's unfinished changes.

After completing a meaningful unit of work:

1. Create a new report under `coop_repo/reports/` using `coop_repo/REPORT_TEMPLATE.md`.
2. Do not overwrite an older report.
3. Update `coop_repo/LATEST.md` to point to the new report.
4. Record validation results and unresolved risks honestly.

The files in `coop_repo/` are coordination records, not substitutes for source code or design documents.
