# Attention Samples

Use these pages to validate the first attention receiver model.

Open in order:

1. `raw.html`: screenshots without annotation.
2. `annotated.html`: same screenshots with predicted attention clusters.
3. `genre_arena_annotated.html`: live iframe annotation shell for the local `/genre_arena/` page.

Samples:

- `steam_home.png`: strong head-signal storefront example.
- `itch_games.png`: dense individual-reading game list example.
- `sudoku_home.png`: puzzle intent with cookie/ad interference.

`password_game.png` was captured but excluded from the sample pages because the page was blocked by a security verification screen.
