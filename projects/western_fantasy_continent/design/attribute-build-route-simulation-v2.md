# Attribute System v2 Build Route Simulation

本报告测试 `attribute_system_v2_candidate.md` 的加点路线空间。

重要边界：本次不改职业基础数值。每个角色以现有 `hp / power / armor / skills` 为基底，只额外叠加 10 点大属性。

实验边界：`berserker` 和 `assassin` 使用新增实验技能套装进行验证；旧技能没有删除，也没有替换正式默认套装。

## 测试方法

- 每个职业生成最多 10 种 10 点加点路线：主属性、副属性、主副混点、纯生存、纯速度、错属性红队、全分散。
- 对手也不是裸数值，而是对应职业的 `7主/3副` 标准路线。
- 每条路线打所有职业的 1v1，每个对局跑 8 个固定 seed。
- 有效路线定义：平均胜率不低于 35%，且距离本职业最佳路线不超过 12 个百分点。

## 当前代理换算

| 大属性 | 本次测试换算 |
| --- | --- |
| 武力 | 每点 `physicalPower +2.35`，`hp +3` |
| 坚韧 | 每点 `hp +14`，`receivedHealing +1.2%`，不增幅吸血 |
| 敏捷 | 每点 `attackSpeed +5.2%`，`effectResist +0.5%` |
| 奥术 | 每点 `magicPower +2.65`，`skillHaste +0.6%` |
| 节律 | 每点 `skillHaste +2.0%`，`effectPower +4%` |
| 韧性 | 每点 `armor +0.5`，`effectResist +1.2%` |

说明：当前正式战斗仍没有完整的物理/法术分离、DOT 增幅、控制强度公式。本次只给模拟器增加了默认不生效的测试字段，用来近似攻速、急速、效果强度、效果抗性。
