# Attribute Growth Prototype Test Report

本报告测试 `attribute_growth_prototype_v1.md` 的大属性加点体验。测试脚本只克隆单位并临时换算属性，不修改正式战斗数值。

## 测试方法

- `5级主属性`：4 点主属性，测试是否能明显打赢 1 级自己。
- `10级主属性`：10 点主属性，测试是否完成第一阶段成长。
- `10级 vs 两个1级`：检查成长是否过强。
- `路线多样性`：每个职业用 `main10 / main7Sec3 / split5 / sec10 / off10` 打其他职业标准路线。
- `队伍测试`：把整队按不同路线加点，观察同一流派是否有不同路线表现。

当前战斗系统还没有完整攻速、暴击、魔抗、治疗强度等公式，所以测试脚本使用代理换算，把未来小属性折算成临时 `hp / power / armor`。这只用于验证方向。

## 最终候选结论

采用最终候选：`attempt-8`。

仍需关注：
- 毒刃刺客 5级主属性成长不明显 (20%)
- 赤狮狂战 非主副属性 10非主副 排名过高
- 灰契术士 有效路线不足 (1)

## 修订后的属性方向

| 大属性 | 定位 | 测试版落地 | 设计原因 |
| --- | --- | --- | --- |
| 力量 | 物理压制 | 攻击中等、生命、破盾/破甲 | 输出带生命副产物，避免纯攻击极端 |
| 体魄 | 生存承载 | 高生命、防御、受治疗 | 让承伤路线能利用治疗/护盾 |
| 敏捷 | 频率机动 | 攻速、移动、闪避，极少攻击 | 防止敏捷同时成为高攻速高攻击 |
| 技巧 | 精准机制 | 命中、标记、少量暴击，极少攻击 | 防止力量+技巧变成高攻高爆 |
| 意志 | 守护抗压 | 治疗、护盾、魔抗、生命 | 支援属性也有生存副产物 |
| 奥秘 | 技能异常 | 技能强度、急速、元素、异常 | 法术走循环和状态，不走暴击乘算 |
| 统御 | 团队协同 | 团队增益、光环、控制、少量生命 | 强化队伍而不是个人主C |

## 发现的问题

- 当前正式战斗公式缺少攻速、暴击、魔抗、治疗强度等派生小属性，测试只能用代理值，不能当最终平衡。
- 部分职业的路线差异来自 `power/hp/armor` 代理，未来接入真实公式后必须重跑。
- 队伍测试显示某些流派路线差异仍不够强，需要后续用装备/技能关键词放大副玩法。
- 非主副属性目前被压低到不太可能成为最优，这是安全的，但以后装备词条可能重新打开 off-build，需要红队测试。

## 下一步建议

1. 你先验收这版是否符合“可感增长 + 2-3条可选方向”。
2. 如果方向通过，再写 `attribute-growth-sim.js` 正式版，接入任务表和报告输出。
3. 再决定是否把攻速、暴击、治疗强度、护盾强度这些小属性逐步接进正式战斗公式。
4. 最后再设计装备部位词条池。
