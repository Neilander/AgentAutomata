# 火毒装备成长过猛诊断

生成时间：2026-07-01

## 结论

火法和毒队成长过猛，主要不是单件装备数值单纯太高，而是三层因素叠在一起：

1. 火毒技能结构本身是非线性的：铺状态、扩散、死亡触发、引爆都会互相放大。
2. 装备映射同时给了生存、技能急速、效果强度/法强，让火毒更容易多跑出一轮关键技能。
3. 当前水桶里缺少足够的后期效果抗性/清状态/打断窗口，所以火毒一旦成型，很多队伍只能被滚雪球。

## 拆解测试

测试方式：不改正式数据，只对指定角色临时加 bonus，然后让对应预设队打新的 100 队增强水桶。

### 火法 fireBurst

基础胜率：0.57

| 临时 bonus | 胜率 | 观察 |
| --- | ---: | --- |
| +140 生命 | 0.75 | 活到爆发窗口后胜率明显提高 |
| +30 法强 | 0.75 | 有效，但不是最夸张 |
| 技能急速 x1.25 | 0.88 | 单项提升最大 |
| 效果强度 x1.25 | 0.69 | 只抬 DOT 不够爆 |
| 法强 + 急速 | 0.89 | 接近急速主导 |
| 效果强度 + 急速 | 0.89 | 仍然是急速主导 |
| 综合 bonus | 0.97 | 接近封顶 |

解释：火法不是单纯“火伤太高”，而是技能急速让双法师更快铺火、更早扩散、更早流星。生命/护甲让法师不在第一轮爆发前死掉。于是装备从线性加伤害，变成了多打一轮完整爆发。

### 毒队 poisonBloom

基础胜率：0.71

| 临时 bonus | 胜率 | 观察 |
| --- | ---: | --- |
| +140 生命 | 0.75 | 有提升，但不夸张 |
| +30 法强 | 0.75 | 有提升，但不是核心 |
| 技能急速 x1.25 | 0.83 | 很强 |
| 效果强度 x1.25 | 0.79 | 比火法更吃效果强度 |
| 法强 + 急速 | 0.87 | 强 |
| 效果强度 + 急速 | 0.88 | 强 |
| 综合 bonus | 0.93 | 接近封顶 |

解释：毒队本身基础已经偏高。它同时吃技能循环和效果强度：更快铺毒、更快引爆、死亡后扩散、引爆后保留并再铺。装备一旦同时给急速和效果强度，毒层收益会跨过关键阈值。

## 正式装备跑出的实际 bonus

增强水桶 8 队刷装模拟末期，火毒常见实际 bonus：

### 火法

- magicPowerAdd：约 40 到 63
- skillHasteMult：约 1.10 到 1.19
- effectPowerMult：约 1.12 到 1.41
- maxHpAdd：约 151 到 331
- armorAdd：封顶到 20

这说明火法装备不是只加输出，它还拿到了相当多生存。对火法这种有明确爆发窗口的队伍，生存等价于“保证能开出爆发”。

### 毒队

- magicPowerAdd：约 20 到 49
- skillHasteMult：约 1.15 到 1.31
- effectPowerMult：固定容易到 1.18 上限
- maxHpAdd：常到 118 到 145
- armorAdd：约 9 到 26

毒队的问题更像“成型后太稳”：它既有持续输出，又通过生命/护甲撑过启动期，同时急速让启动期变短。

## 关键原因排序

1. 技能急速对火毒收益过高。
   - 火法：急速单项把胜率从 0.57 推到 0.88。
   - 毒队：急速单项把胜率从 0.71 推到 0.83。

2. DOT/状态队伍同时拿到了启动、生存、结算三种收益。
   - 生存保证进入 payoff。
   - 急速提高铺状态频率。
   - 效果强度提高 DOT tick 和引爆质量。

3. 火毒技能带有天然扩散和死亡触发。
   - 火：燃烧目标死亡后触发火种余爆。
   - 毒：中毒目标死亡后扩散毒层。
   - 这会导致装备提升不是只提升单体，而是提升滚雪球速度。

4. 对抗侧不足。
   - 增强水桶虽然更均匀，但并没有系统性提供效果抗性、净化、打断、快速切后排等反火毒能力。

## 不是主要原因的部分

1. 不是单纯法强太高。
   - 法强单项有效，但没有急速那么夸张。

2. 不是水桶太弱。
   - 新水桶已经能把基础队压到 30 左右，但火毒装备后跨过关键阈值。

3. 不是某一件装备独立超标。
   - 问题更像词条组合和技能结构共振。

## 调整建议

### 优先调整

1. 火毒对技能急速设置折损。
   - 不是全局削弱技能急速，而是对 DOT/铺场/引爆类技能使用更低的 haste 转换。
   - 例如普通技能吃 100% skillHaste，状态铺层技能只吃 60%-75%。

2. 火毒装备不要同时高概率获得三件套收益。
   - 三件套收益指：生存、急速、效果强度。
   - 可以保留两项，第三项通过低权重或高稀有度控制。

3. 给高层水桶加入明确反制组件。
   - 效果抗性。
   - 净化。
   - 开局切后排。
   - 打断或延迟大招。

### 次级调整

1. 火法死亡溅射和流星之间做一点互斥或递减。
2. 毒队引爆后的保留/再铺可以稍微降低，避免一次启动后永动。
3. 装备评分里对 `dotAmp + fireAmp/poisonAmp + skillHaste` 的同套组合加更强风险惩罚。

## 暂不建议

1. 不建议直接砍火法/毒队基础数值。
   - 基础队当前不是离谱，离谱发生在装备成长后。

2. 不建议全局砍技能急速。
   - 牧师等弱成长队伍也吃急速，全局砍会误伤弱队。

3. 不建议只降低装备掉落品质。
   - 会让所有流派成长都变慢，但不能解决火毒的共振问题。

## 下一步验证

推荐下一轮开独立任务：

1. 做三组装备规则 A/B：
   - A：火毒技能急速折损。
   - B：装备生成降低 `skillHaste + dotAmp + archetypeAmp` 同套概率。
   - C：增强水桶加入反火毒单位。

2. 每组重新跑 8 队刷装模拟。

3. 目标：
   - 火法末期不超过 0.85 到 0.90。
   - 毒队末期不超过 0.85 到 0.90。
   - 牧师/暗影不能被误伤，最好仍有轻微提升。
   - 最弱队末期不低于 0.45。
