# Western Fantasy Continent Plan

1. Build a local auto-battler combat prototype.
2. Define reusable class rules for tank, warrior, assassin, and mage.
3. Let the user configure left and right teams with 1 to 4 units each.
4. Add simple movement, attack timing, targeting, damage, and win conditions.
5. Prototype a pseudo-Spine skeleton so character art, gear, facing, and reusable actions can expand without rewriting combat rendering.
6. Add a local backend so tools can save files into the project instead of only using browser cache.
7. Build a richer stat, equipment, relic, and combat-event foundation for the auto-battler.
8. Later connect units to persistent NPC attributes, dialogue identity, equipment, and continent factions.
