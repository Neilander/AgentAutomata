# Pseudo Spine Skeleton Demo

Open `index.html` to view the character skeleton prototype.

This is not a real Spine runtime yet. It is a browser-native prototype that copies the important production ideas:

- bones
- slots
- replaceable attachments
- mirrored facing
- reusable animation states
- class-based silhouettes

The goal is to prove the expandable structure before importing or authoring real Spine assets.
